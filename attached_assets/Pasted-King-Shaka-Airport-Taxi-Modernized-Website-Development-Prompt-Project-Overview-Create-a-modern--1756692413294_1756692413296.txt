King Shaka Airport Taxi - Modernized Website Development Prompt
Project Overview
Create a modern, responsive website for King Shaka Airport Taxi, a premium taxi service operating in KwaZulu-Natal, South Africa since 2010. The website should maintain existing branding while implementing contemporary design principles and enhanced user experience.
Brand Identity & Design Guidelines
Company Information
•	Business Name: King Shaka Airport Taxi
•	Tagline: "Reliable & Affordable Rides in KZN"
•	Established: 2010 (serving with pride since 2010)
•	Location: KwaZulu-Natal, South Africa
•	Primary Service Area: King Shaka International Airport and throughout KZN
Color Scheme (maintain existing branding)
•	Primary Colors: Professional blues and whites (corporate/trustworthy)
•	Accent Colors: Gold/yellow highlights (taxi industry standard)
•	Background: Clean whites with subtle greys
•	Text: Dark greys/blacks for readability
•	Call-to-Action: Prominent blues or golds
Logo Requirements
•	Maintain existing "King Shaka Airport Taxi" branding
•	Incorporate airplane/taxi/transportation elements
•	Professional, clean design
•	Scalable for various screen sizes
•	If logo redesign needed, use: airplane icon, taxi silhouette, or road elements

Technical Requirements
Technology Stack
•	Frontend: HTML5, CSS3, JavaScript (ES6+)
•	Framework: React.js or Vue.js (recommended)
•	Styling: Tailwind CSS or modern CSS frameworks
•	Icons: Font Awesome or Lucide React
•	Animations: Framer Motion or CSS animations
Responsive Design
•	Mobile-first approach
•	Breakpoints: 320px, 768px, 1024px, 1440px
•	Touch-friendly buttons (minimum 44px)
•	Optimized images for different screen densities
Performance Optimization
•	Fast loading times (under 3 seconds)
•	Optimized images (WebP format)
•	Minified CSS/JS
•	Lazy loading for images
•	SEO optimized
